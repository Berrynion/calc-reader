# calc-reader
基于华米手表（Amazfit zeppOS≥3.0） "calc-reader"  一个隐藏阅读器 · 科学计算器:一个伪装成计算器的小说阅读器。 表面是能用的科学计算器，输入密码按 " = "入隐藏书架
隐藏阅读器 · 伪装计算器（Amazfit zeppOS）

**初始密码123456

适配手表:Amazfit Balance / T-Rex 3 / Cheetah Pro / GTR 4 / Active 2（圆屏）
**仅Balance测试通过其它设备仅做简单适配不知道是否可以正常使用

v1.0  基础版
功能：
- 伪装计算器 + 密码进隐藏书架
- 阅读器：点击/表冠翻页、字号调节、跳页、进度记忆
- 内置书 + 手机端在线上传（下载直链）
- 圆屏安全区适配

v2.0  基本重构
功能：

🧮计算器:
- UI美化
- 三页科学计算器：基础 / 函数(π、ln、x²、√、阶乘…) / 三角(sin、cosh、反函数…)
- 第四页单位换算页：长度/重量/温度/容积，点一下即换算
- 历史记录：上滑查看，可回填结果

📖阅读器：

安装：

1.打开手机端zepp应用-设置-关于-连续点击zepp图标直到开启开发者模式

2.返回zepp应用设备页-通用-开发者模式-右上角" + "-扫一扫扫描对应设备QR码

上传（手表端书架有说明教程）:

1.获取小说下载直链（建议使用OPENLIST) **UTF8编码

2.打开开发者模式界面-小程序-计算器设置

3.输入下方书名与TXT直链

4.保持手表处在书架页，点击开始上传，耐心等待即可  **上传过程手机不要切换其它应用
